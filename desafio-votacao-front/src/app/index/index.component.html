Bem vindo ao aplicativo de votação!

Instruções:

  1. Comece cadastrando associados no menu
    • O CPF precisa ser válido para funcionar.
    • Clique no botão CPF para saber se o CPF é válido e apto para votar.
    • Clique no botão Gerar CPF para gerar um CPF válido.
    • Ao salvar o associado, ele aleatoriamente atribuirá o status de apto para votar ou não.
  2. Depois você poderá cadastrar pautas no menu
    • Ao iniciar a sessão de votação, defina quantos minutos ela ficará aberta.
    • Em seguida, proceda a declarar os votos. Restrições:
      • Um associado só pode votar uma vez por sessão.
      • O associado precisa ser apto para votar.
